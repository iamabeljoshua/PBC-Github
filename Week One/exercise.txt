1. Easy: Write a program which will find all such numbers which are divisible by 7 but are not a multiple of 5 
between 2000 and 3200 (both included).

The numbers obtained should be printed in a comma-separated sequence on a single line.

2. Easy: Write a program that accepts a sentence and calculates the number of letters and digits. 
Suppose the following input is supplied to the program:

hello world! 123

Then, the output should be:

LETTERS 10 

DIGITS 3
